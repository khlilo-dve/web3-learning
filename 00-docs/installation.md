# 安装步骤文档

本文件记录大一 Web3 项目环境搭建和工具安装的步骤。

## 1. 安装 VS Code

### 下载
- 官方网址: [https://code.visualstudio.com/](https://code.visualstudio.com/)
- 选择 Windows 版本，点击下载

### 安装
1. 双击下载的安装包
2. 按照默认选项安装（建议勾选 “Add to PATH” 和 “Add Open with Code”）
3. 安装完成后打开 VS Code
4. 验证：
   - 打开终端（Ctrl+`）
   - 输入 `code --version`，显示版本号则成功

## 2. 安装 Python

### 下载
- 官方网址: [https://www.python.org/downloads/](https://www.python.org/downloads/)
- 下载最新稳定版本（Windows Installer）

### 安装
1. 双击安装包
2. 勾选 **Add Python to PATH**
3. 点击安装，等待完成
4. 验证：

   python --version
